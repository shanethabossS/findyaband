See your band live on the road.

Find My Band lets authorized drivers securely transmit their GPS location so NCC, event teams and spectators can see where a Carnival band is in real time.

Private test build. Android only. You may need to allow installs from your browser / “unknown apps” in settings.

For NCC & event teams

This prototype is built for Trinidad & Tobago Carnival. The goal: a simple, low-cost way to know where each band is on the parade route using drivers’ phones.

  • 1 Drivers install the app and log in with a tracker account.
  • 2 They select their band and tap Start transmitting.
  • 3 The map dashboard shows each band’s live position and updates as they move.

This page hosts the Android APK download only. The admin map / dashboard is run separately for testing.

GPS based Secure logins Low data usage Built with Flutter & Supabase

How to install & test (Android)

  • 1 Tap the Download Android APK button above and wait for the file to finish downloading.
  • 2 Open the APK from your notifications or Downloads folder. If prompted, allow your browser (Chrome, etc.) to install unknown apps.
  • 3 Open the app, log in with a test tracker account, select a band and tap Start transmitting.
  • 4 Keep the app running while you drive. The GPS will keep transmitting even if the screen is off, as long as the app is not closed.

This is an early prototype. Performance, battery usage and safety flows will be refined with feedback from NCC, band leaders and drivers.
